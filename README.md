
# PHP Specification Pattern

Реализация паттерна **Specification** на PHP для гибкой и расширяемой работы с бизнес-правилами.

## Описание

Паттерн Specification позволяет инкапсулировать бизнес-правила в отдельные классы и комбинировать их между собой с помощью логических операций (AND, OR, NOT).

Это избавляет от сложных `if/else` конструкций и делает код:

- читаемым
- расширяемым
- переиспользуемым
